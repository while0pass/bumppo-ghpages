• Версия 0.8

    Отображение числа найденных соответствий, области поиска и частично условий поиска на странице результатов. Отображение контекста и номера записи для результатов. Возможность переключаться между разными видео. Доделаны реальные свойства единиц вокального, окуломоторного и мануального каналов.

    Реализованы все основные типы свойств единиц поиска. Начальная интеграция с серверной частью. Показ выбранных видеофрагментов с выключением просмотра при достижении конца фрагмента.

Активация/деактивация кнопки поиска в зависимости от того, готов ли запрос и стали запрос новым по сравнению с последним поиском.

    В дереве запроса сверстаны отрицательные единицы поиска и условия на расстояние между единицами.

  • Версия 0.3

  • Версия 0.2

    Начерно сверстаны основные разделы: выбор подкорпуса, дерево запроса, результаты. Клиентская навигация. Минимум интерактива: переключение между разделами, проигрывание видео с начала выбранного щелчком фрагмента.
